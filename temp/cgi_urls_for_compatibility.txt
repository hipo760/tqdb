CRITICAL PARAMETERS TO SUPPORT:
===============================

Query Parameters (common across query URLs):
--------------------------------------------
- symbol                    # Trading symbol
- BEG                       # Begin date/time 
- END                       # End date/time
- csv=1                     # Return CSV format
- timeoffset                # Time offset parameter
- MOSTHAVEBEG/MUSTHAVEBEG   # Validate date range (typo variations exist)

Import Parameters:
------------------
- html=1                    # Return HTML format
- importTicket              # Import job identifier
- sym                       # Symbol form data
- tzConv                    # Timezone conversion
- tzSelect                  # Timezone selection
- file                      # File upload

Edit Parameters:
----------------

- epochFloatEnd             # End time range

Configuration Parameters:
-------------------------

- confVal                   # Configuration value

---------------------
- application/json          # For JSON responses
- text/csv                  # For CSV downloads
- text/html                 # For HTML responses
- text/plain                # For plain text

---------------------
- CSV downloads need Content-Disposition header with filename
- Support gzip compression (Content-Encoding: gzip)
- HTML responses support meta refresh for status monitoring
- Support ^^ prefix for multi-leg symbols
- Handle multipart form data for CSV imports

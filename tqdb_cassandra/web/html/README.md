# Static Web Files

HTML, CSS, and JavaScript files for the TQDB web interface.

## Files

### HTML Pages
- `index.html` - Main landing page
- `esymbol.html` - Symbol management
- `edata.html` - Data editor
- `i1min.html` - Import minute bars
- `symsummery.html` - Symbol summary
- `tqalert.html` - Alert configuration
- `tqdate.html` - Date range queries

### Stylesheets
- `style.css` - Main application styles
- `jquery-ui.css` - jQuery UI theme
- `jquery-ui-timepicker-addon.css` - Timepicker styles

### JavaScript Libraries
- `jquery-2.1.4.min.js` - jQuery core
- `jquery-ui.min.js` - jQuery UI widgets
- `jquery-ui-sliderAccess.js` - Slider accessibility
- `jquery-ui-timepicker-addon.js` - Datetime picker

### Assets
- `ui-icons_*.png` - jQuery UI icon sprites

All files are served by Apache from `/var/www/html/`.
